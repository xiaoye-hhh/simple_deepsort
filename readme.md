# 简单的追踪算法
看不懂deep_sort的操作,写一个简单的算法: 主要是取消了卡尔曼滤波

# 代价矩阵的计算方法
1. 求出feature_matrix
2. 求取目标的质心,判断相机是否抖动
   1. 如果大幅抖动,直接计distance_matrix为0
   2. 否则,进行简单计算distance_matrix,只考虑方框中心点是否位于100x100的范围内
      1. 在此范围内,则distance_cost计为0
      2. 否则,根据丢失跟踪的帧数计算出一个cost,丢失越久,cost越小
3. 将俩距离举证相加,得到最终的距离矩阵
